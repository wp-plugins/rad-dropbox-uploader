== Description ==

Easily allow both administrators & front end users to upload files to specific folders on your Dropbox account using administration interface or easily customizable front-end shortcode!

== Installation ==

1. Unzip contents of “rad-dropbox-uploader.zip”.
2. Upload folder “rad-dropbox-uploader” to your “/wp-content/plugins” directory.
3. Activate plugin under WordPress administration: Dashboard > Plugins.
4. Enjoy!
